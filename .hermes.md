# OPC Ecommerce — 多 Agent 跨境电商 Listing 生成系统

## 项目概述
这是一个多 Agent 系统，用于生成 Amazon/Etsy/eBay 商品 Listing。
项目根路径：`C:\Users\nicho\OPC_ecommerce`

## 如何确定你的身份
**如果你是独立的 Agent 对话窗口，你的 AGENT.md 定义了你是谁。**
使用以下命令加载你的身份：
```
read_file: agents/[你的Agent名]/AGENT.md
```
例如 Visual Agent：`read_file agents/visual/AGENT.md`

## Boss Agent（你的上级）
Boss Agent 是主控 Orchestrator。
它会：分配任务、路由事件、管理闸门、通知你什么时候开始工作。
你不需要知道 Boss Agent 是谁——它会在合适的时候通知你。

## 铁律（所有 Agent 必须遵守）
1. 每个 Listing 最多 2 个变体维度
2. 合规检测仅通过 Dify 执行
3. 任何越权写入 → 清除脏数据 → 重跑
4. 三振出局：连续失败 3 次 → 中断 → 通知 Boss
5. shared/databases/ 对子 Agent 只读
