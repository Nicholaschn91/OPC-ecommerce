## 2. 选轨锚点

### 快速排除清单
- SPU 含 "handmade wood / bamboo decor / artisan craft / wood carving" + 家居装饰载体 → **直接锁定 Track C**
- 品类为木质/竹制装饰品，且无 A/B2/D 信号 → **默认 Track C**

### 仲裁规则
- **C1 工艺叙事**: 品类命中 Track K 且存在手工/匠人叙事信号 → **升轨至 Track C**
- **C4 纯装饰无实用**: 纯观赏性木制品，无功能性收纳/使用场景 → **默认 Track C**
