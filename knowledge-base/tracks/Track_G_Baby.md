# Track G — Baby & Kids

> **赛道代号**: G
> **赛道名称**: Baby & Kids
> **版本**: V1.0
> **更新**: 2026-07-13

---

## 1. 赛道定义

| 维度 | 定义 |
|------|------|
| **核心品类** | 婴童用品 / 儿童玩具 / 母婴配件 |
| **核心动机** | 安全保护 / 亲子互动 / 发育支持 |
| **核心情感** | 强安全合规要求 / 马卡龙配色强制 |
| **典型 SPU** | 婴儿围嘴、出牙胶、学步车 |

---

## 2. 选轨锚点

- SPU 含 "baby / infant / toddler / kids / nursery" + 婴童载体 → **直接锁定 Track G**
- 品类为婴童用品，且无 A-F 信号 → **默认 Track G**

---

## 3. 文案风格指令

| 维度 | 要求 |
|------|------|
| **叙事模式** | 安全第一、发育支持、亲子温情 |
| **关键词部署** | T4 词放安全认证/材质/年龄段 |
| **材质翻译** | 必须标注 BPA-free、无邻苯、食品级、通过 ASTM/EN71 |
| **视觉调性** | 马卡龙配色强制 / 圆角设计 / 柔和光影 |

---

## 4. 绝对禁令

| 编号 | 禁令 |
|------|------|
| **M1** | **严禁小部件窒息风险**（<3岁必须标注） |
| **M2** | 必须通过 CPC/CPSC/ASTM F963/EN71 认证 |
| **M3** | 配色仅限马卡龙/莫兰迪色系 |
| **M4** | 禁止成人化营销语 |
