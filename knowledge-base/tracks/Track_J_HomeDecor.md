---

## 1. 赛道定义

| 维度 | 定义 |
|------|------|
| **核心品类** | 墙面艺术品 / 装饰画 / 挂件 |
| **核心动机** | 空间美化 / 色彩搭配 / 艺术收藏 |
| **核心情感** | 色彩还原精度要求高 / 无礼赠主导 |
| **典型 SPU** | 帆布画、金属墙饰、装饰时钟 |

---

## 2. 选轨锚点

- SPU 含 "wall art / canvas print / wall decor / poster" + 墙面装饰载体 → **直接锁定 Track J**
- 品类为墙面艺术，且无 A-I 信号 → **默认 Track J**

---

## 3. 文案风格指令

| 维度 | 要求 |
|------|------|
| **叙事模式** | 色彩还原、空间氛围、艺术收藏价值 |
| **关键词部署** | T4 词放风格/尺寸/题材/色彩 |
| **材质翻译** | 画布/墨水/相框材质→档次/耐久/色彩还原 |
| **视觉调性** | 色彩还原精度高 / 场景化空间展示 |

## 5. 绝对禁令

| 编号 | 禁令 |
|------|------|
| **M1** | 色彩还原必须准确（ΔE < 1.5） |
