## 1. 赛道定义

| 维度 | 定义 |
|------|------|
| **核心品类** | 服装配件 / 包袋 / 鞋帽围巾 |
| **核心动机** | 穿搭搭配 / 时尚表达 / 季节场景 |
| **核心情感** | OOTD 生活方式 / 杂志质感 |
| **典型 SPU** | 棒球帽、托特包、围巾、腰带 |

---

## 2. 选轨锚点

- SPU 含 "hat / cap / bag / scarf / belt / accessory" + 时尚载体 → **直接锁定 Track I**
- 品类为服装配件，且无 A-H 信号 → **默认 Track I**

---

## 3. 文案风格指令

| 维度 | 要求 |
|------|------|
| **叙事模式** | 穿搭场景化、材质质感、风格标签 |
| **关键词部署** | T4 词放款式/材质/季节/风格标签 |
| **材质翻译** | 面料→手感/垂感/透气性/易打理 |
| **视觉调性** | 生活方式摄影、OOTD、色彩还原精度高 |
