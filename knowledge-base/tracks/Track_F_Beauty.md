---

## 1. 赛道定义

| 维度 | 定义 |
|------|------|
| **核心品类** | 美妆 / 护肤 / 食品接触类 / 个护仪器 |
| **核心动机** | 自我提升 / 日常护理 / 健康生活 |
| **核心情感** | 皮肤安全感 / 成分信任 / 仪式感 |
| **典型 SPU** | 面部清洁仪、食品级硅胶模具、护肤冰箱 |

---

## 2. 选轨锚点

- SPU 含 "skin contact / food grade / cosmetic / beauty device" + 皮肤/食品接触载体 → **直接锁定 Track F**
- 品类为美妆/个护/食品接触，且无 A-E 信号 → **默认 Track F**

---

## 3. 文案风格指令

| 维度 | 要求 |
|------|------|
| **叙事模式** | 成分透明、安全合规、使用体验 |
| **关键词部署** | T4 词放成分/认证/安全标准 |
| **材质翻译** | 必须标注 FDA/欧盟食品接触/化妆品法规合规 |
| **视觉调性** | 实验室级、洁净感、成分可视化 |

## 5. 绝对禁令

| 编号 | 禁令 |
|------|------|
| **M1** | **严禁医疗宣称**（cure/treat/heal/prevent） |
